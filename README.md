# Nuxt-Studio-Test

Prueba de concepto para un repositorio con Nuxt Studio editor.

## Stack

- [Nuxt 4](https://nuxt.com/)
- [Nuxt UI v4](https://ui.nuxt.com/)
- [Nuxt Content v3](https://content.nuxt.com/)
- [Nuxt Studio](https://nuxt.studio) (módulo open-source auto-hospedado)

## Instalación (con pnpm)

```bash
pnpm install
pnpm dev
```

## Estructura

Este proyecto usa la template oficial **Docs** de Nuxt UI v4, adaptada para funcionar con el módulo `nuxt-studio`.

## Edición con Nuxt Studio

En desarrollo, accede a `/studio` en tu servidor local para probar el editor. En producción (SSR), configura la autenticación OAuth con GitHub para permitir edición visual directamente desde el navegador.
